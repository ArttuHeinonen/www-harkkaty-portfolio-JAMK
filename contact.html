Dokumentti

Suunnitelma

Tavoitteena oli luoda portfolio sivusto, jonka pystyisi lähettämään työnantajalle. Portfolioon pitäisi tulla tietoja itsestä sekä omasta työkokemuksesta alalla. Jos aikaa jäisi, niin tekisin integroidun java applet pelin yhdelle sivuista.

Sivuston toteutus

Sen sijaan, että olisin luonut sivut alusta asti, päätin etsiä netistä valmiin html pohjan, jota käyttää portfolion pohjana. Valitsin pohjakseni Zerotype mallin sivustolta https://www.freewebsitetemplates.com

Käytetyt tekniikat

Sivustossa käytettiin vain HTML skriptiä ja CSS tyyliskriptejä. Javaskriptiä, eikä jQueryä käytetty.

Itseisarvio

Sivuston tekemiseen meni jonkin verran aikaa, mutta en prioirisoinut tätä kurssia tällä opintojaksolla. Siitä johtuen tekniikoiden taidonnäytteitä ei tässä sivulla näekkään. Paremman siistimisen, huolellisemman tekstin ja kuvien muokkaamisen jälkeen kuitenkin kehtaisin näyttää tätä sivua portfolionani.
